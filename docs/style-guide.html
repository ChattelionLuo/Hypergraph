Internal guide

Text styles for Project Hypergraph.

Use this page as a copy-and-paste reference when editing headers, paragraphs, cards, buttons, and prediction-page text.

Quick rules

How to edit text safely.

1. Edit only the words first.

Most text is inside an HTML tag. Keep the opening and closing tags unchanged unless you want to change the style.

<h2>Ranking science</h2>

2. Change style by changing the tag or class.

The word after class= applies a reusable style from styles.css.

<p class="eyebrow">Research</p>

Headings

Use large headings sparingly.

h1

Main hero headline.

Use for major sections and subpage cards.

Use inside cards, model summaries, and compact content blocks.
